# Mangalam HDPE Pipes Landing Page

A premium industrial product landing page built from a Figma design using pure HTML, CSS, and JavaScript.

This project focuses on clean UI implementation, responsive layouts, smooth interactions, reusable components, and realistic production-style frontend architecture.

## Project Overview

This landing page was developed as a frontend implementation assignment based on a provided Figma design.

The goal was not only to match the design visually, but also to enhance the experience with polished UI interactions, responsive behavior, reusable components, and production-level frontend structure.

The project includes:

- Responsive navigation
- Interactive product carousel
- Hover zoom effects
- Infinite horizontal marquees
- Technical specification table
- FAQ accordion
- Testimonial slider
- Resource download section
- Global reusable modal system
- Interactive forms
- Toast notifications
- Fully responsive mobile experience
